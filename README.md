# PySnake
Snake game written in Python

![gif](https://github.com/dbrizov/dbrizov.github.io/blob/master/images/project-images/pysnake.gif)

## System Requirements
Just run the .exe, or run it with Python 3+

## Controls
- Movement: **WASD / Arrow Keys**
- Restart: **Enter**
- Toggle Pause: **Space**
- Quit: **Esc**
